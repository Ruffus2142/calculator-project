## Функції


### 1. `add(a, b)`

- **Опис:** додає два числа.
- **Параметри:**
   - `a` — перше число.
   - `b` — друге число.
- **Повертає:** суму чисел.
- **Приклад:**
  ```python
result = add(5, 3)  # 8


### 2. subtract(a, b)

- **Опис:** віднімає друге число від першого.

- **Параметри:**
- `a` — перше число.
- `b` — друге число.

- **Повертає:** різницю чисел.

- **Приклад:**

result = subtract(10, 4)  # 6


### 3. `multiply(a, b)`

- **Опис:** множить два числа.

- **Параметри:**

- `a` — перше число.
- `b` — друге число.

- **Повертає:** добуток чисел.

- **Приклад:**

result = multiply(7, 6)  # 42


### 4. divide(a, b)

- **Опис:** ділить перше число на друге.

- **Параметри:**

- `a` — чисельник.
- `b` — знаменник.

- **Повертає:** результат ділення (або повідомлення про помилку, якщо `b == 0`).

- **Приклад:**

result = divide(20, 5)  # 4
