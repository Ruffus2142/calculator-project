Calculator Project

Це простий навчальний калькулятор, створений для практики роботи з HTML, CSS та JavaScript. Він дозволяє виконувати базові арифметичні операції: додавання, віднімання, множення та ділення.

Встановлення

Склонуйте репозиторій на свій комп’ютер:

git clone https://github.com/ruffus2142/calculator-project.git
cd calculator-project

Приклади використання

Після відкриття сторінки ви можете:

  • Ввести два числа
  • Обрати потрібну операцію
  • Натиснути кнопку "Обчислити"
  • Отримати результат на екрані
